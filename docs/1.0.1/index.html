Descrição

O componente gumga-click-sync-ng bloqueia os botões enquanto seus requisições estõ sendo realizadas, isso evita que o usuário clique mais de uma vez no mesmo lugar antes que um retorno seja feito.

Instalação

// NPM
npm i gumga-click-sync-ng --save

// BOWER
bower install gumga-click-sync-ng --save

//Modulo
angular.module('app', ['gumga.click-sync'])

// NPM
<script src="node_modules/gumga-click-sync-ng/dist/gumga-click-sync.min.js"></script>

//BOWER
<script src="bower_components/gumga-click-sync-ng/dist/gumga-click-sync.min.js"></script>
Usando
  • HTML
  • JavaScript
<button gumga-click-sync="save({name: 'example'})"
        class="btn btn-primary">
        Click me
</button>
angular.module('app', ['gumga.click-sync'])
  .controller('Ctrl', function($scope){

    //SEU METODO DEVE RETORNAR UMA PROMISSE
    $scope.save = function (x) {
        return service(x).then(function (resp) {
            console.log(resp)
        })
    }

  })
Atributos
Atributo Tipo Requerido Descrição
gumga-click-sync Function Sim Use gumga-click-sync ao invés de ng-click, execute uma função que retorne uma PROMISSE, o botão será desabilitado no click do usuário, após a PROMISSE ser resolvida o botão irá habilitar automaticamente.
